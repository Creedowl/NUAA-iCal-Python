## 安装

### PyPI 安装

安装 `NUAAiCal` python 软件包:

```bash
$ pip install NUAAiCal
```

### 源码安装

从源码构建安装:

```bash
$ git clone https://github.com/Triple-Z/NUAA-iCal-Python.git
$ cd NUAA-iCal-Python
$ python3 setup.py install
```

## 启动程序

```bash
$ nuaaical
```

你将会看到课程表日历 `.ics` 文件的导出路径, 你可以将其导入至 Google 日历等日历程序。
