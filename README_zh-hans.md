# NUAA iCal Python

:us: [English](/README.md) | :cn: 简体中文

## 使用方法

安装依赖:

```bash
$ pip install -r requirement.txt
```

启动程序:

```bash
$ python3 main.py
```

> 只支持 python 3+

课程表日历文件将导出到 `target/curriculum.ics`, 你可以将其导入至 Google 日历等日历程序。
