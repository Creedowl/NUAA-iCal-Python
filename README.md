## Quick start

Install dependencies:

```bash
$ pip install -r requirement.txt
```

Start the application:

```bash
$ python3 main.py
```

> ONLY supported by python 3+

Your curriculum will in `target/curriculum.ics`, you can import it to Google 
Calendar etc.
